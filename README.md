# Instrukcje

1. Klonowanie repozytorium, w pożądanej lokalizacji:

`git clone https://github.com/SuperGierki/ProjektSemestralny1.git`

2. Utworzenie wirtualnego środowiska Python:

`python -m venv .venv`

3. Aktywacja wirtualnego środowiska

- Windows:
`.venv\Scripts\activate`

- Linux/MacOS:
`source .venv/bin/activate`

4. Zainstalowanie niezbędnych modułów:

`pip install -r requirements.txt`

5. Utworzenie nowego brancha:

`git branch [nazwa]`

6. Zmiana na nowego brancha:

`git checkout [nazwa]`
